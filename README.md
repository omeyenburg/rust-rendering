# Rust Rendering

Simple rendering in Rust with OpenGL and SDL, focused on flexible usage.

I plan to turn this into a library someday.

## Usage

Clone the repo and run the demo:
```sh
git clone https://github.com/omeyenburg/rust-rendering.git
cd rust-rendering
cargo run --release
```
